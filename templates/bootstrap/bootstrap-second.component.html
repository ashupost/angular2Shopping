Written by Mritunjay Kumar

It is possible to develop all kinds of applications with Java that can work seamlessly together. The power and flexibility of Java is enormous, but being able to develop in Java requires knowledge, insight and craftmanship. In the Java Certified Associate Track, part of the Master Class Java, you will be introduced to the objected oriented design, programming skills, the basic elements of the Java programming language and class design using UML. From the beginning with your craftmanship at the core, you will be challenged to independently use theory in practical cases to develop maintainable and portable object oriented code in Java.
